# Python URL Shortener

This is a simple Python script that shortens any long URL using the TinyURL API.

## Features
- Takes any long URL as input
- Sends HTTP request to TinyURL API
- Returns a short URL
- No API key required

## Requirements
- Python 3.x
- requests module

Install the required module:

pip install requests

## How to Use
Run the script using:

python url_shortener.py

Then enter the long URL when prompted.
